# NOTICE — Propriedade de marca

Este repositório contém o **design system do redesign da Jaé** (tokens,
componentes React Native, telas de exemplo e documentação visual), produzido
como trabalho de design para o projeto.

A identidade visual da Jaé (cores, tipografia e logo) pertence à **Jaé** e seus
respectivos detentores de direitos. Os arquivos aqui derivam desse trabalho de
design para o cliente e seguem o acordo entre as partes.

> A cópia espelhada do site jae.com.br **não faz parte deste repositório** —
> ela é mantida apenas localmente, fora do versionamento, por conter material
> de marca de terceiro.
